Detector de Comportamientos Sospechosos

Subir Video para Análisis

Suba un video para detectar comportamientos sospechosos utilizando análisis de postura con MediaPipe.

Comportamientos detectados:

  • Acercamiento disimulado para posible cartereo o sustracción de objetos de valor
  • Ocultación de manos o manipulación de bolsillos
  • Miradas frecuentes y cambios bruscos de dirección
Sobre este proyecto

Este sistema analiza videos para identificar comportamientos potencialmente sospechosos utilizando MediaPipe para el seguimiento de poses y movimientos corporales.

El sistema marcará los momentos sospechosos después de que la actividad haya persistido durante al menos 3 segundos, reduciendo significativamente los falsos positivos.
